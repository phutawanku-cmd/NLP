## วิธีการรันโปรแกรม (How to Run)

เนื่องจากระบบมีการใช้ฟังก์ชัน `input()` เพื่อรับค่าจากผู้ใช้แบบ Real-time จึงต้องรันผ่าน Terminal/Command Line เท่านั้น

1. เปิด Terminal ในโฟลเดอร์ที่เก็บไฟล์
2. พิมพ์คำสั่ง: `python eliza_bot.py` (หรือ `python3 eliza_bot.py` สำหรับ Mac/Linux)
3. พิมพ์ข้อความโต้ตอบกับบอทได้ทันที
4. พิมพ์ `quit` เมื่อต้องการออกจากโปรแกรม
